# 📋 List Row Separators — Completed
A SwiftUI example showing how to customize, hide, and style List row separators.

---

## 🤔 What this is
This is the completed version of the List Row Separators project. It demonstrates how to use `.listRowSeparator()`, `.listRowSeparatorTint()`, and related modifiers to control separator visibility and color on a per-row basis in a SwiftUI List. The finished project you can reference alongside the tutorial.

## ✅ Why you'd use it
- **Separator visibility** — Shows how to hide or show separators on individual rows with `.listRowSeparator(.hidden)`
- **Separator tinting** — Demonstrates `.listRowSeparatorTint(_:)` to apply custom colors to row dividers
- **Edge control** — Covers controlling which edges (top/bottom) show a separator per row

## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/NDCSwift/ListRowSeperatorsCompleted.git
cd ListRowSeperatorsCompleted
```

### 2. Open in Xcode
Double-click `ListRowSeperators.xcodeproj`.

### 3. Set Your Development Team
In Xcode: **TARGET → Signing & Capabilities → Team** — select your team.

### 4. Update the Bundle Identifier
Change `com.example.MyApp` to a unique reverse-domain ID.

## 🛠️ Notes
- This is the completed version. For the starter template, see the `ListRowSeperatorsTemplate` repo.
- If you see a code signing error, verify Team and Bundle ID are set.
